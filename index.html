Pipeline / Architecture
Extract frames from the 3D film with scene detection and RIFE interpolation→Multi-layer depth-aware segmentation (U²-Net / SAM / ISNet) with LaMa background inpainting→Cluster characters via CLIP embeddings + HDBSCAN + UMAP, then interactive human review→Auto-caption with BLIP2 using a trigger-token / identity-tag strategy→Prepare the dataset and pose data (MediaPipe/DWPose on CPU, parallel to GPU)→Train SDXL LoRA, ControlNet pose, or AnimateDiff via Kohya_ss / Diffusers→Evaluate checkpoints with LPIPS/FID and compare in ComfyUI workflows
Key Capabilities

Config-driven orchestrator

YAML/TOML multi-stage orchestrator with resource monitoring, progress tracking, and checkpoint resume.

3D-tuned segmentation

Lower alpha (0.15) and blur (80) thresholds plus depth-aware matting for smooth 3D shading and lighting.

Automatic character grouping

CLIP + HDBSCAN + UMAP cluster characters with smaller cluster sizes, refined via interactive review.

Identity-aware captioning

BLIP2 captions led by trigger token and identity tags (age/gender) to prevent feature drift in identity LoRAs.

CPU-parallel pose prep

MediaPipe/DWPose pose extraction at 60+ FPS on 32 threads, running alongside GPU LoRA training.

ComfyUI evaluation

LoRA checkpoint comparison, ControlNet pose control, and video generation tests with LPIPS/FID metrics.
